\section{Question EC 34}
A digital transmission system uses a (7,4) systematic linear Hamming code for transmitting data over a noisy channel. If three of the messageâ€“codeword pairs in this code $(m_i ; c_i)$ where $c_i$ is the codeword corresponding to the $i^th$ message $m_i$ , are known to be (1100;0101100), (1110;0022220) and (0110 ; 1000110), then which of the following is a valid codeword in this code?
\begin{enumerate}
\item[A]0110100 
\item[B]1011010
\item[C] 0001011 
\item[D]1101001
\end{enumerate}
\section{Solution}
Given code is systematic linear hamming code of order (7, 4) \\ 
\\
Given message and code word pairs are \\
1100; 0101100\\
1110; 0011110\\
0110; 1000110\\ 
\\
The code word is of the form\\
c = p1 p2 p3 m1 m2 m3 m4 \\
where
\begin{align}
P1 = d1 \oplus d2 \oplus d4\\
P2 = d2 \oplus d3 \oplus d4\\
P3 = d1 \oplus d2 \oplus d3\\
\end{align}
The code word which satisfies this pattern is 0001011
